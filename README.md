🤖 SOPHIA – Voice Assistant

SOPHIA is a simple Python-based voice assistant that can listen to voice commands and perform basic tasks like opening websites and speaking responses.

🚀 Features
Voice recognition using microphone
Text-to-speech response
Opens websites like Google, YouTube, Instagram, LinkedIn
Simple command-based system

🛠️ Tech Used
Python
SpeechRecognition
pyttsx3
Groq API
requests
webbrowser

📦 Installation
git clone https://github.com/SAKSHI10987/Sophia-voice-assistant.git
cd sophia
pip install -r requirements.txt
▶️ Run the Project
python main.py
